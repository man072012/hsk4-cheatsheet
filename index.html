📖 ترتيب الجمل — الأسئلة ٥٦ إلى ٦٥
١
إذا شفت 但 / 可是 / 不过 / 却 في جملة → ما تجي أولاً أبداً
١٠٠٪ ٧ مرات
هالكلمات معناها "لكن". اللي قبلها هي الفكرة الأصلية واللي بعدها التناقض. حطها ثانية أو ثالثة.
مثال — H41327 سؤال ٥٧
B: 钱虽然能买到很多东西
A: 但世界上还有很多是钱买不到的
C: 例如生命、爱情、友谊和时间
الترتيب: B → A → C
٢
إذا شفت 所以 / 这样 / 因此 في جملة → حطها أخيراً
١٠٠٪ ٤ مرات
هالكلمات معناها "لذلك / هكذا". النتيجة دايم آخر شي.
مثال — H41328 سؤال ٥٩
B: 我办了新闻手机报服务
C: 每天都能收到经济、社会和国际等方面的新闻短信
A: 这样我就可以及时了解国内外发生的大事了
الترتيب: B → C → A
٣
إذا شفت نداء مثل 各位 / 欢迎 / 人们常说 / 爷爷 → حطها أولاً
١٠٠٪ ٧ مرات
الجمل الافتتاحية (نداء، ترحيب، مقولة شائعة) دايم تبدأ الفقرة.
مثال — H41329 سؤال ٥٨
B: 各位乘客，为了保证您和他人的安全
A: 请您按照"先下后上"的顺序上下车
C: 并注意脚下，照顾好老人和孩子
الترتيب: B → A → C
٤
إذا شفت 它 / 它们 / 这些 / 那儿 / 这样的 في جملة → ابحث عن الاسم الأصلي وحطه قبلها
١٠٠٪ ٧ مرات
الضمير ما يجي من العدم. لازم الشي اللي يشير له يكون مذكور قبل.
مثال — H41329 سؤال ٦١
B: 写日记是一个很好的习惯
C: 它可以帮我们记住过去发生的事情
A: 给我们留下很多美好的回忆
它 تشير لـ 写日记 → لازم B قبل C — الترتيب: B → C → A
٥
إذا شفت 不但/不仅/不光 في جملة و 还/而且/更 في ثانية → "不但" قبل "还"
١٠٠٪ ٤ مرات
مثال — H41327 سؤال ٦٠
B: 我提高汉语水平的方法就是坚持看中文报纸
C: 这样不但能学到很多新的词语
A: 还能扩大知识面
الترتيب: B → C → A
٦
إذا شفت 只有 في جملة و 才 في ثانية → "才" تجي أخيراً
١٠٠٪ ٢ مرات
"只有...才" = "فقط إذا... حينها فقط". الشرط أولاً والنتيجة أخيراً.
مثال — H41328 سؤال ٦١
A: 学习时，不光要知道答案是什么
C: 还要弄清楚答案究竟是怎么得来的
B: 只有这样，才能把问题真正弄懂
الترتيب: A → C → B
٧
إذا شفت 既然 في جملة و 那么 في ثانية → "既然" أولاً
١٠٠٪ ١ مرة
"既然...那么" = "بما أن... إذن". الحقيقة أولاً ثم الاستنتاج.
مثال — H41327 سؤال ٦٤
B: 两个人既然决定共同生活
C: 那么，只有浪漫和新鲜感是不够的
A: 懂得互相信任和尊重才是最重要的
الترتيب: B → C → A
٨
إذا شفت 随着 في جملة → حطها أولاً
١٠٠٪ ٢ مرات
"随着" = "مع تغير...". تحدد السياق والخلفية فتجي أولاً.
مثال — H41330 سؤال ٦٠
B: 这几天植物园特别热闹，随着天气变暖
A: 那儿的花儿都开了，非常漂亮
C: 吸引了很多人前去参观
الترتيب: B → A → C
٩
إذا شفت 吸引了很多人/游客 في جملة → حطها أخيراً
١٠٠٪ ٣ مرات
جذب الناس = نتيجة نهائية لشيء جميل أو مميز.
١٠
إذا شفت 例如 / 比如 / 顺便 في جملة → حطها أخيراً
١٠٠٪ ٢ مرات
المثال أو الفعل الإضافي (顺便 = بالمناسبة) يجي بعد الفكرة الرئيسية.
११
إذا شفت 本来 في جملة و 后来 في ثانية → "本来" أولاً
١٠٠٪ ١ مرة
"本来" = "في الأصل" ← "后来" = "لاحقاً". الأصل قبل التغيير.
مثال — H41328 سؤال ٦٢
C: 这次招聘会本来是由小李负责的
B: 由于他突然生病住院了
A: 后来就交给我来做了
الترتيب: C → B → A
१२
إذا شفت 相反 في جملة → ابحث عن الجملة المعاكسة وحطها قبلها
١٠٠٪ ١ مرة
"相反" = "على العكس". لازم يكون في جملة قبلها تقول العكس.
مثال — H41330 سؤال ٥٩
C: 做自己喜欢的事，即使再困难，也不会觉得辛苦
A: 相反，做自己不喜欢的事
B: 即使再简单也会觉得很累
الترتيب: C → A → B
१३
إذا شفت 意思是 في جملة → حطها بعد المصطلح اللي تشرحه
١٠٠٪ ١ مرة
"意思是" = "معناها هو...". لازم يكون المصطلح أو الفكرة مذكورة قبلها.
مثال — اختبار ٦ سؤال ٥٧
A: 有人说现在社会上流行"快餐文化"
B: 意思是无论做什么事
C: 人们都希望用最短时间来完成
المصطلح(快餐文化) في A → الشرح(意思是) في B → الترتيب: A → B → C
१४
إذا شفت 可惜的是 في جملة → حطها أخيراً أو بعد الإيجابي
१٠٠٪ १ مرة
"可惜的是" = "المؤسف هو...". تجي بعد ذكر شي إيجابي ثم التحسر عليه.
مثال — اختبار ٦ سؤال ٦٢
C: 昨天的京剧演出实在太精彩了
A: 看完之后，我的心情特别激动
B: 可惜的是喜欢看京剧的年轻人越来越少了
إيجابي(C) → ردة فعل(A) → تحسر(B) — الترتيب: C → A → B
१५
إذا شفت 于是 في جملة → ما تجي أولاً
१٠٠٪ ٣ مرات
"于是" = "وهكذا/فـ" — نتيجة سردية لأحداث متتابعة (مختلفة عن 所以 اللي نتيجة منطقية). معناها حصل حدث ثم حدث آخر بالتتابع.
مثال — سؤال ٣
B: 他从来不主动和别人说话
A: 于是大家都以为他是一个骄傲的人
C: 其实他只是有点儿害羞
الترتيب: B → A → C
१६
إذا شفت 以为 في جملة → بعدها تجي 其实 / 原来 / 后来 للتصحيح
१٠٠٪ ٣ مرات
"以为" = "ظن خطأ" — يجي بعدها تصحيح الاعتقاد بكلمة 其实 أو 原来 أو 后来.
مثال
Structure: False belief(以为) → Correction(其实/原来)
१७
إذا شفت 其实 في جملة → ما تجي أولاً أبداً
१٠٠٪ ٤ مرات
"其实" = "في الحقيقة" — تصحح اعتقاد خاطئ، لازم يكون في اعتقاد قبلها.
१८
إذا شفت 换句话说 / 就是说 في جملة → ما تجي أولاً
१٠٠٪ ٢ مرات
"换句话说" = "بعبارة أخرى" — تعيد صياغة شي قبلها.
१९
إذا شفت 平时 / 往往 / 常常 / 每天都 في جملة → غالباً أول جملة
१٠٠٪ ٥ مرات
هالكلمات تدل على العادة والروتين — تكون تمهيد قبل الاستثناء أو التفصيل.
مثال — سؤال ١
C: 平时我骑自行车上下班
A: 可是今天起晚了
B: 所以就打车来公司
الترتيب: C → A → B
२०
إذا شفت 甚至 في جملة → ما تجي أولاً (تصعيد)
१٠٠٪ ٢ مرات
"甚至" = "حتى إن / لدرجة إن" — تضيف معلومة أقوى من اللي قبلها.
مثال — سؤال ١١
A: 很多人都曾经试着对梦进行解释
C: 有些人甚至专门写过这方面的书
B: 可惜到现在仍然没有一个科学的说法
الترتيب: A → C → B
२१
إذا شفت 难怪 في جملة → حطها أخيراً
१٠٠٪ ٢ مرات
"难怪" = "لا عجب / ما ألوم" — خلاصة واستنتاج بعد معرفة السبب.
مثال — سؤال ١٧
A: 这家餐厅的菜味道不错
B: 服务也很周到
C: 难怪生意这么好
الترتيب: A → B → C
२२
إذا شفت 如果 / 要是 في جملة و 就 في ثانية → "如果" قبل "就"
१٠٠٪ ٣ مرات
"如果...就" = "إذا...فـ" — الشرط أولاً والنتيجة بعده.
مثال — سؤال ١٢
B: 邀请信我已经写好了，你再仔细检查一下
C: 如果没问题的话
A: 明天就可以发出去了
الترتيب: B → C → A
२३
إذا شفت 也 في جملة → ما تجي أولاً أبداً (إضافة)
१٠٠٪ ٥ مرات
"也" = "أيضاً" — تضيف على شي مذكور قبلها.
२४
إذا شفت 以前 / 从前 / 曾经 في جملة → غالباً أول جملة
१٠٠٪ ٣ مرات
كلمات الماضي تفتح الكلام — تذكر حالة سابقة ثم يجي التغيير.
مثال — سؤال ١٠
C: 我记得，以前村子的旁边有一条小河
A: 河水不深，非常清
B: 清得可以看见河底的水草和成群的小鱼
الترتيب: C → A → B
२५
إذا شفت 是因为 في جملة → ما تجي أولاً (تفسير)
१٠٠٪ ٢ مرات
"是因为" = "والسبب في ذلك هو" — تفسر نتيجة مذكورة قبلها.
२६
إذا شفت 要想 في جملة و 就/首先 في ثانية → "要想" قبل "就/首先"
१٠٠٪ ٢ مرات
"要想...就" = "إذا تبي... لازم أولاً" — الهدف ثم الوسيلة.
२७
إذا شفت 一...就 — "بمجرد أن...حصل" → تسلسل سريع
१٠٠٪ ٢ مرات
V1 一 V2 就 = بمجرد ما حصل الأول، الثاني صار فوراً.
مثال — سؤال ٧
C: 100年前，塑料一出现
B: 就受到人们的普遍欢迎
A: 当时的人们没想到它会给环境带来严重的污染
الترتيب: C → B → A
💡 خطوات الحل السريعة — ترتيب الجمل
١. استبعد الجمل اللي ما تقدر تكون أولاً (但、可是、所以、于是、其实、也、甚至)
٢. ابحث عن القواعس المزدوجة (虽然...但是، 不但...还، 如果...就)
٣. اربط الضمائر بأصلها (它، 这些، 那儿)
٤. اقرأ الترتيب بسرعة وتأكد من المعنى المنطقي
✍️ ترتيب الكلمات — الأسئلة ٨٦ إلى ٩٥
💡 القاعدة الذهبية
أول شي حدد: ١) الموضوع (فاعل) ٢) الفعل الرئيسي ٣) أي كلمة مفتاحية (把/被/得/比/离). الكلمة المفتاحية تحدد لك الترتيب تلقائياً.
A
إذا شفت 把 → الصيغة ثابتة:
١٠٠٪ ٦ مرات
Subject + 把 + Object + Verb + Result/Complement
الفعل لازم يكون معاه مكمل (得/到/进/完/好/了) — ما يجي لحاله أبداً.
مثال — H41327 سؤال ٩١
姐姐 把 房间 收拾得很整齐
فاعل(姐姐) + 把 + مفعول(房间) + فعل+مكمل(收拾得很整齐)
مثال — H41329 سؤال ٩٠
请 把 空瓶子 扔进垃圾桶
طلب + 把 + مفعول(空瓶子) + فعل+اتجاه(扔进垃圾桶)
B
إذا شفت 被 → الصيغة ثابتة:
१٠٠٪ ٣ مرات
Affected + [Adverb] + 被 + Agent + Verb + Result
عكس 把 — الشي المتأثر يجي أول الجملة.
مثال — H41330 سؤال ٨٦
那盒饼干 被 儿子 吃光了
متأثر(那盒饼干) + 被 + فاعل(儿子) + فعل+مكمل(吃光了)
C
إذا شفت 得 → الصيغة ثابتة:
१٠٠٪ ٥ مرات
Verb + 得 + Degree Description
‫"得" لازم تلتصق بالفعل مباشرة — ما يجي شي بينهم.
مثال — H41329 سؤال ٨٨
他 咳嗽得 越来越厉害
فاعل(他) + فعل+得(咳嗽得) + وصف(越来越厉害)
مثال — H41332 سؤال ٩٣
他的普通话 说得 不太标准
D
إذا شفت 这个/这条/这台/这种/这次/这些 → حطها أول الجملة
१٠٠٪ ٩ مرات
هذي إشارة للموضوع الرئيسي. الموضوع دايم أول شي في الجملة الصينية.
مثال — H41329 سؤال ٨٧
这种儿童牙膏 十分受欢迎
مثال — H41330 سؤال ٩٠
这两个词语的用法 有什么区别？
E
إذا شفت 比 → الصيغة ثابتة:
१٠٠٪ ٣ مرات
A + 比 + B + Verb/Adjective + Difference (last)
مثال — H41329 سؤال ٩٥
今年公司的收入 比 去年 增加了一倍
A(今年收入) + 比 + B(去年) + فعل(增加了) + الفرق(一倍)
F
إذا شفت 是 و 的 مع بعض → إطار تأكيد:
१٠٠٪ ٤ مرات
Topic + 是 + ... + 的
مثال — H41327 سؤال ٩٢
这次调查 是 由李经理负责 的
G
إذا شفت 离 → الصيغة ثابتة:
१٠٠٪ ٢ مرات
Place A + 离 + Place B + Distance
مثال — H41329 سؤال ٩٣
北京 离 上海 大约1084公里
H
إذا شفت 将 / 将于 → حطها قبل الفعل مباشرة
١٠٠٪ ٢ مرات
"将" = "سوف" (رسمي). تجي بعد الفاعل وقبل الفعل.
مثال — H41330 سؤال ٩٥
飞机 将于 15分钟后 降落在首都机场
I
إذا شفت 难道 → حطها أول الجملة + علامة ؟
१٠٠٪ ١ مرة
"难道" = "أيعقل أن...؟" — سؤال بلاغي استنكاري.
مثال — H41329 سؤال ٨٩
难道 你没有先调查一下？
J
إذا شفت مكان + 里/上 مع 有 → المكان أولاً (جملة وجودية)
१٠٠٪ १ مرة
Place phrase+里/上 + 有 + Number + Thing
عكس العربية — ما تقول "يوجد شجرتان في الحديقة" بل "في الحديقة يوجد شجرتان".
مثال — H41329 سؤال ٩٢
张教授家的院子里 有 两棵苹果树
المكان(院子里) + 有 + العدد(两棵) + الشي(苹果树)
K
إذا شفت 让 → الصيغة ثابتة:
१٠٠٪ १ مرة
Cause/Thing + 让 + Person + Verb/Adjective
"让" = "يخلي / يجعل". الشي اللي يسبب يجي أول، والشخص المتأثر بعد 让.
مثال — اختبار ٦ سؤال ٩١
妈妈的鼓励 让 孩子 不再害怕
السبب(妈妈的鼓励) + 让 + المتأثر(孩子) + النتيجة(不再害怕)
L
إذا شفت فعل+مفعول مع مدة زمنية → كرر الفعل:
१٠٠٪ १ مرة
Subject + V+O + V了 + Duration
الأفعال المركبة (离合词) مثل 弹钢琴 ما تقدر تحط المدة بعدها مباشرة — لازم تكرر الفعل.
مثال — اختبار ٦ سؤال ٨٦
我 弹钢琴 弹了 三年了
فاعل(我) + فعل+مفعول(弹钢琴) + فعل مكرر+了(弹了) + المدة(三年了)
M
إذا شفت 连 في الكلمات و 都/也 → 连 + Object + 都/也 + Verb
१٠٠٪ ٢ مرات
"连...都" = "حتى...كلها/بعد" — تأكيد وتعجب.
Subject + 连 + Object + 都/也 + Verb
مثال — H41003 سؤال ٩٤
难道你 连这个规定 都不知道？
N
إذا شفت 竟然 في الكلمات → Subject + 竟然 + Verb
१٠٠٪ ٢ مرات
"竟然" = "بشكل مفاجئ" — ظرف يأتي قبل الفعل.
مثال — H41004 سؤال ٨٧
教授 竟然把这次机会放弃了
O
إذا شفت 由...组成 في الكلمات → Subject + 由 + Parts + 组成
१٠٠٪ ٢ مرات
"由...组成" = "يتكون من" — تركيب.
مثال — H41110 سؤال ٨٦
这篇文章 由三部分 组成
P
إذا شفت 深受...欢迎/喜爱 → Subject + 深受 + Group + 的 + 欢迎/喜爱
१٠٠٪ ٢ مرات
"深受" = "يحظى بـ" — تقدير عميق.
مثال — H41004 سؤال ٩١
那位著名的演员 深受观众 的喜爱
Q
إذا شفت 以...为 → Subject + 以 + A + 为 + B
१٠٠٪ १ مرة
"以...为" = "يأخذ...كـ" — تحديد أساس أو معيار.
مثال — H41004 سؤال ٩٢
普通话 以北京语音 为标准音
R
إذا شفت 不得不 → Subject + 不得不 + Verb
१٠٠٪ ٢ مرات
"不得不" = "مضطر/لا بد" — اضطرار.
مثال — H41002 سؤال ٩٣
重感冒让他 不得不请假休息

🔗 أدوات الربط المهمة — مرجع سريع
أدوات "لكن" (التناقض)
但是 可是 不过 然而 而 却 相反
→ ما تجي أولاً
أدوات السبب والنتيجة
因为...所以 因此 由于 于是
→ 所以/因此/于是 ما تجي أولاً
أدوات الشرط
如果...就 要是...就 只要...就 只有...才
→ الشرط أولاً ثم النتيجة
أدوات "بالرغم من"
虽然...但是 尽管...但是 尽管...可是 尽管...却
→ 虽然/尽管 أولاً
أدوات التصعيد
不但...还 不仅...而且 甚至
→ من الأقل للأكثر
أدوات التصحيح
以为...其实 本来...后来 原来
→ اعتقاد خاطئ ثم تصحيح
🎯 نصائح سريعة للاختبار
الأسئلة ٥٦-٦٥: ابدأ بالمسح البصري
قبل ما تقرأ المعنى، امسح الجمل الثلاث بعينك وابحث عن أي كلمة مفتاحية (但، 所以، 它، 各位...). إذا لقيت وحدة، طبق القاعدة مباشرة — هذا يقلل الاحتمالات من ٦ إلى ٢ أو ١.
الأسئلة ٨٦-٩٥: حدد الفعل أولاً
في الكلمات المبعثرة، حدد الفعل الرئيسي أولاً. إذا كان معاه 把 أو 被 أو 得 أو 比، الصيغة تفرض نفسها تلقائياً. إذا ما في كلمة مفتاحية، طبق: فاعل + زمان + مكان + فعل + مفعول.
⚠️ فخ شائع: موضع الظروف مع 把/被
✅ 他已经把作业做完了
❌ 他把作业已经做完了
✅ 他没办法把这个问题解释清楚
✅ 这些货物暂时被存放在邮局
الظروف (已经/都/刚/没办法/暂时) تجي قبل 把/被 — مو بعدها
⏱️ إدارة الوقت في الاختبار
الأسئلة ٥٦-٦٥ (ترتيب الجمل): خذ ٣٠ ثانية لكل سؤال. لو ما قدرت حددت في ١٥ ثانية → انتقل. الأسئلة ٨٦-٩٥ (ترتيب الكلمات): خذ ٢٥ ثانية لكل سؤال — أسرع شوية لأنها ما تحتاج قراءة.
